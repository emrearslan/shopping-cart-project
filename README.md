# Shopping Cart Project
Implement simple an e-commerce shopping cart.

## Running
To run project:
```
./gradlew build
java -jar build/libs/shopping-cart-project-0.0.1-SNAPSHOT.jar
```

To run all tests:
```
./gradlew test
```

## Rest
Swagger Url is `http://localhost:9070/shopping-cart-project/swagger-ui.html`.
